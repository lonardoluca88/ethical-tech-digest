Ethical Tech Digest - Guida all'installazione

Questa guida ti aiuterà a installare il widget Ethical Tech Digest sul tuo sito web o piattaforma Kajabi.

Kajabi
Sito HTML
Configurazione

Installazione su Kajabi

1 Accedi a Kajabi e vai al pannello di amministrazione.

2 Scegli la pagina in cui vuoi inserire il widget.

3 Aggiungi un blocco HTML personalizzato alla pagina.

4 Copia il codice di incorporamento qui sotto e incollalo nel blocco HTML.

✓ Copiato!<div id="ethical-tech-digest"></div>
<script src="https://leonardo2030.entourage-di-kryon.it/lovablenews/widget/ethical-tech-digest.js"></script>

5 Salva le modifiche e pubblica la pagina.

Installazione su sito HTML

1 Apri il file HTML della pagina in cui vuoi inserire il widget.

2 Aggiungi il seguente codice nel punto in cui desideri visualizzare il widget:

✓ Copiato!<div id="ethical-tech-digest"></div>
<script src="https://leonardo2030.entourage-di-kryon.it/lovablenews/widget/ethical-tech-digest.js"></script>

3 Salva il file e carica le modifiche sul tuo server web.

Consiglio: Inserisci il codice all'interno di un elemento container per controllare meglio la larghezza e l'aspetto del widget.

Opzioni di configurazione

È possibile personalizzare il widget utilizzando attributi dati nell'elemento contenitore:

✓ Copiato!<div id="ethical-tech-digest" 
  data-theme="light" 
  data-categories="ai,robotics,biotech">
</div>

Opzioni disponibili:

  • data-theme: Imposta il tema del widget ("light" o "dark")
  • data-categories: Specifica le categorie da visualizzare inizialmente (separate da virgola)
Importante: L'ID "ethical-tech-digest" è obbligatorio e non deve essere modificato.
